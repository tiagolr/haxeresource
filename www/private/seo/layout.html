About

Haxe resource is a wiki site that gathers haxe information such as articles and tutorials.

It contains features like article voting and tag groups to make it easier to find content. Because a lot of articles are scattered in blogs, it also features external articles with links only.

This site is built using meteorjs and haxe and is open source.

For more information, questions or suggestions head over to the forum. If you have tutorials to write or know some that should be listed here, feel free to signup and start posting.
